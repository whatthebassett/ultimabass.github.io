Bassett's Style Guide For Website Design

Color Guide

The following colors should be taken into consideration when building the site. There should be no differences in color beyond what is listed here.

Pugent Purple
HEX Code Value: #6644FF
Pretty Pink
HEX Code Value: #FF00A0
Blue Light
HEC Code Value: #FFFFF0

The color guides will be used in tandom for most of the fonts and HEADER tags. The colors will also represent tables and DIVs with color.

Lighter colors will mostly be used for table cells if needed and could be links, but it maybe too light.
