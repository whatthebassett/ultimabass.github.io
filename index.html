    HTML Tags & Definitions
    HTML Tag HTML Description
    <table> This is the beginning of the table element. This the element you should use when beginning to make a table.
    <thead> THEAD should be the next element you use under table. This is basically a metatag so you can ensure that all headers are fit snuggly in the table correctly.
    <tbody> TBODY will be used for the actual content of the table. Always make sure this goes BENEATH the THEAD element.
    <th> This element will create the actual header of the cell. Think of this as a title of the column, simuilar to work's excel spreadsheets.
    <tr> TR will creat the row of the cell you want to make. This comes after the TH element and will create the physical row of the table for the data. This should always go BEFORE < TD >.
    <td> This is the element that will contain the actual data of your table. You know...the important stuff. This comes last when entering data for the table.
    <pre> The "pre" tag stands for "preformatted" and will not execute any HMTL or CSS property that's coded inside of it.

    CSS Properties & Definitions
    CSS Properties CSS Description
    Position Position can be used in a multitude of ways. It can be used to move elements around in order "position" them around a webpage. The following can be used in-junction with this property:


    • Static: This is the default value. This position keeps the element part of the normal flow of the document. This will only needed to be spelled out when you need to reset the element when it's nested in another element with a different position property listed.

    • Absolute: This will remove the element from the webpage's design flow and will give the element no space in the layout. This element used in tandem with the TOP, LEFT, RIGHT, and BOTTOM positioning properties.

    • Fixed: The attribute will place the element out of the normal flow of the document. It creates no space for the element in the page layout and its positioning is based on the containg block of the document.

    • Absolute: This will remove the element from the webpage's design flow and will give the element no space in the layout. This element used in tandem with the TOP, LEFT, RIGHT, and BOTTOM positioning properties.
    Float Using float will make it so make the element move over to the position where you want that element to be on the page. You can use float with the following common attributes:
    • left
    • right
    • inline-start
    • inline-end
    • none

    Float will also allow text to move around images and other elements. This makes it appear as if the element is trying to move out of the way of other elements.
